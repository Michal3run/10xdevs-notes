# m2l5 — Linki

## Tools

- [Git worktree documentation](https://git-scm.com/docs/git-worktree.html) — oficjalny opis `git worktree add`, listowania, modelu wielu working trees w jednym repo.
- [Keep Claude working toward a goal](https://code.claude.com/docs/en/goal) — `/goal`: warunek zakończenia, ewaluator, tryb headless; porównanie z `/loop` i Stop hooks.
- [Cursor Agents](https://cursor.com/product) — orkiestrator multi-agent w Cursorze.
- [Antigravity (Google)](https://developers.googleblog.com/build-with-google-antigravity-our-new-agentic-development-platform/) — IDE z Manager view do orkiestracji wielu agentów.
- [Superset](https://superset.sh/) — orkiestrator do równoległych CLI agentów w izolowanych worktrees.
- [Conductor](https://www.conductor.build/) — orkiestrator równoległych Claude Code/Codex z jednym widokiem review.

## Reference

- [Ralph Wiggum technique (Geoffrey Huntley)](https://ghuntley.com/ralph/) — bashowa pętla `while true` jako prymitywny ewaluator; tło dla zrozumienia, dlaczego `/goal` jest ulepszoną wersją tego samego pomysłu.

## Prework

- [Prework 3.3 — Cykl życia wątku i zarządzanie kontekstem](https://platforma.przeprogramowani.pl/external/10xdevs-3-prework/pl/11) — strategia `Isolate` z preworku tu staje się konkretna: izolujesz katalog, branch, sesję agenta i kontekst zadania.
