# m1l3 — Linki

## Tools

- [Configure permissions (Claude Code)](https://code.claude.com/docs/en/permissions) — składnia `permissions.allow/deny/ask`, kolejność ewaluacji, wzorce `Bash(...)`.
- [Choose a permission mode (Claude Code)](https://code.claude.com/docs/en/permission-modes) — pełna lista trybów (`default`, `plan`, `auto`, `bypassPermissions`) i ostrzeżenia wokół `--dangerously-skip-permissions`.
- [Development containers (Claude Code)](https://code.claude.com/docs/en/devcontainer) — referencyjny `devcontainer.json`: nie-rootowy user, ograniczona sieć — warunki YOLO mode.
- [npm audit (npm CLI v10)](https://docs.npmjs.com/cli/v10/commands/npm-audit) — pięć poziomów ważności + semantyka exit codes (przyda się przy `verification.md`).
